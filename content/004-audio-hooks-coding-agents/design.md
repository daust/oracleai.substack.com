# Episode 004: Never Miss a Claude Code Prompt Again — Add Audio Cues to Your AI Workflow

**Status:** Draft
**Goal:** Set up two audio hooks in Claude Code so you hear an alert the moment the AI needs your input and a victory sound when it finishes — without watching the screen.
**Audience:** Anyone using Claude Code (macOS or Windows)

## Requirements

- Claude Code (any recent version with hooks support)
- An audio file for each cue (`.mp3`, `.wav`, `.aiff`) — macOS users can generate them with the built-in `say` command; everyone else can use any sound file they like
- macOS or Windows (playback commands differ; both are covered)
